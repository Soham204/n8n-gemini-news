# gemini-news-agent

A n8n workflow that fetches news, summarizes it using Google Gemini, and sends the digest to a Discord channel.

## Features
- Scheduled news fetch (HTTP Request)
- AI summarization via Google Gemini
- Automatic posting to Discord
- Lightweight JavaScript parsing nodes
